# Fantom

Fantom — это легковесный интерпретируемый язык программирования с C-подобным синтаксисом, созданный для обучения, экспериментов и быстрой разработки.

## 🚀 Запуск

1. Открой папку **Fantom**
2. Перейди в `dist`
3. Запусти `fantom_app.exe`

Готово.

## 📌 Пример кода

```fa
функция главная {
    отобразить("Привет, мир!")
}
```

## ⚠️ Дисклеймер

Fantom является экспериментальным проектом и использует выполнение кода через механизм Python (`exec`).

Это означает, что:

* любой выполняемый код фактически является Python-кодом
* запуск недоверенных или сторонних `.fa` файлов может быть небезопасен
* проект **не предназначен для использования в защищённых или продакшн-средах**

❗ Используйте Fantom **только на свой страх и риск**.

Разработчик не несёт ответственности за:

* любой прямой или косвенный ущерб
* потерю данных

* **любые файлы формата `.fa`, открываемые или запускаемые пользователем, включая их содержимое и последствия их выполнения**
